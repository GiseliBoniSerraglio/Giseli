EXPLICAÇÃO

A adição de duas matrizes é um processo simples que envolve a soma dos elementos correspondentes de ambas as matrizes. Podemos seguir passos para realizar a adição de matrizes. Primeiro verifique se as duas matrizes têm as mesmas dimensões. A adição de matrizes só é possível se as duas matrizes tiverem a mesma quantidade de linhas e colunas. Após verificado, crie uma nova matriz que terá a mesma dimensão que as matrizes originais. Asim adicione os elementos correspondentes de cada matriz para obter o elemento correspondente na nova matriz. Ou seja, some o elemento na mesma posição em ambas as matrizes e coloque o resultado na mesma posição na nova matriz. Repita o passo 3 para todos os elementos na matriz, adicionando-os aos elementos correspondentes na outra matriz. Quando todos os elementos tiverem sido adicionados, a nova matriz conterá a soma das duas matrizes originais.
